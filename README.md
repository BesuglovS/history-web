# 🏛️ История России - Тестирование

Сайт для практики решения задач по истории. Позволяет ученикам проходить тесты по различным темам, а администратору — управлять заданиями и просматривать статистику.

## 🚀 Быстрый старт

### Размещение на хостинге с PHP

Просто разместите файлы на любом хостинге с поддержкой PHP:

1. Скопируйте все файлы в корневую папку сайта (public_html)
2. Убедитесь, что папка `data` доступна для записи (chmod 755 или 775)
3. Сайт будет доступен по адресу вашего домена

**Структура URL:**

- **Главная страница**: `http://your-domain.com/`
- **Админ-панель**: `http://your-domain.com/admin.html`

### Локальный запуск с Node.js

```bash
npm install
npm start
```

Сайт будет доступен по адресу: **http://localhost:3000**

### Пароль администратора

По умолчанию: `admin123`

Можно изменить в файле `data/config.json`.

## 📚 Функционал

### Для учеников

- ✅ Выбор темы для тестирования
- ✅ **Два режима работы:**
  - **📖 Практика** — вопросы по одному, мгновенная проверка, можно завершить в любой момент
  - **📝 Тест** — фиксированное количество вопросов (5/10/15/20), все вопросы на одной странице
- ✅ Три типа вопросов:
  - Один правильный ответ
  - Несколько правильных ответов
  - Открытый ответ (текст)
- ✅ Ввод имени (опционально)
- ✅ Просмотр результатов с разбором ошибок
- ✅ Прогресс-бар
- ✅ Подсветка правильных/неправильных ответов

### Для администратора

- ✅ Авторизация
- ✅ Загрузка заданий из текстовых файлов
- ✅ Редактирование тем и вопросов
- ✅ Удаление тем
- ✅ Просмотр статистики:
  - Общее количество тестов
  - Количество учеников
  - Средний результат
  - Статистика по темам
- ✅ Просмотр результатов с фильтрацией:
  - По имени ученика
  - По теме
  - По дате
  - Сортировка по столбцам

## 📄 Формат файла с заданиями

Создайте текстовый файл (.txt) со следующей структурой:

```
ТЕМА: Название темы

ВОПРОС: Текст вопроса?
1. Вариант 1
2. Вариант 2
3. Вариант 3
4. Вариант 4
ОТВЕТ: 2

ВОПРОС: Вопрос с несколькими правильными ответами?
1. Вариант 1
2. Вариант 2
3. Вариант 3
ОТВЕТ: 1, 3

ВОПРОС: Открытый вопрос?
ОТВЕТ: Правильный ответ
```

### Правила:

- `ТЕМА:` — название темы (один раз в начале файла)
- `ВОПРОС:` — текст вопроса
- `1.`, `2.`, ... — варианты ответов (нумерация с 1)
- `ОТВЕТ:` — номер(а) правильных ответов через запятую
- Для открытых вопросов просто укажите ответ после `ОТВЕТ:`

## 🔧 API

### Публичные endpoints

| Метод | Путь                           | Описание                   |
| ----- | ------------------------------ | -------------------------- |
| GET   | `/api/topics.php`              | Список тем                 |
| GET   | `/api/questions.php?topic=xxx` | Вопросы темы (без ответов) |
| POST  | `/api/submit.php`              | Отправить ответы           |

### Администраторские endpoints

| Метод  | Путь                           | Описание        |
| ------ | ------------------------------ | --------------- |
| POST   | `/api/admin/login.php`         | Авторизация     |
| GET    | `/api/admin/results.php`       | Все результаты  |
| GET    | `/api/admin/stats.php`         | Статистика      |
| GET    | `/api/admin/topics.php?id=xxx` | Тема с ответами |
| POST   | `/api/admin/upload.php`        | Загрузить файл  |
| PUT    | `/api/admin/topics.php?id=xxx` | Обновить тему   |
| DELETE | `/api/admin/topics.php?id=xxx` | Удалить тему    |
